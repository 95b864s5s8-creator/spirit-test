### 用神选择原则

**身强原则**：身旺能扛，宜"克泄耗"
- 最旺=同类（我太固执）→ 克我（修剪）或 我生（泄秀）或 我克（耗力）
- 最旺=生我（被溺爱）→ 我生（泄秀输出）或 我克（耗力担当）
- 最旺=我生（透支过度）→ 克我（收敛约束）或 同类（帮身回血）
- 最旺=克我（高压摧残）→ 身强能扛，可通关化解（生我或我生）
- 最旺=我克（欲望反噬）→ 克我（约束纪律）或 生我（补充底气）

**身弱原则**：身弱扛不住，宜"生扶"
- 最旺=同类（外强中干）→ 生我（补充能量）或 同类（稳固根基）
- 最旺=生我（输入窒息）→ 同类（帮身自立）或 我克（主动出击打破被动）
- 最旺=我生（掏空自己）→ 生我（回血补充）或 同类（帮身分担）
- 最旺=克我（被压垮）→ 生我（通关化解）或 同类（帮身扛压）
- 最旺=我克（扛不起的欲望）→ 生我（给底气）或 同类（帮身分担）

### 推导优先级

1. 通关优先：当最旺和日主直接冲突（克我/我克）时，优先找能通关的五行
2. 生克平衡：身强用克泄，身弱用生扶
3. 避免极端：不用"同类×同类"组合（越帮越偏），除非身弱+最旺=克我（需要帮手）

### 通关关系（关键）

通关 = 找到能同时连接日主和最旺的五行：
- 日主木 + 最旺金（金克木）→ 水（金生水→水生木）✅
- 日主木 + 最旺水（水多木浮）→ 土（克水筑坝）或 火（泄木消耗水气）✅
- 日主木 + 最旺火（木生火泄气）→ 土（木克土→火生土，转化输出）✅
- 日主木 + 最旺土（木克土费力）→ 火（木生火→火生土，顺畅转化）✅

通用公式：
- **克我通关**：日主 ← 最旺克 → 通关 = 生我者（最旺通关生我）
- **我克通关**：日主 → 克最旺 → 通关 = 我生者（我通关生最旺）
- **我生通关**：日主 → 生最旺 → 通关 = 我克者（日主通关→最旺）
- **生我通关**：日主 ← 生最旺 → 通关 = 克我者（切断生源或修剪）

用神从25种变50种，但守护神矩阵仍然是 5用神×4阵营=20格。

因为身强身弱影响的是"用神五行"的选择，选出来之后仍然只有木/火/土/金/水五种结果。20格矩阵不需要改。

**实际影响**：同一个日主+最旺组合，身强和身弱可能得出不同用神→匹配不同守护神。这提升了精准度。

举例：
- 木型人+最旺金+身强 → 用神=水 → 水×阵营 → 观音/龙王等
- 木型人+最旺金+身弱 → 用神=水 → 同上（这条碰巧一致）

- 火型人+最旺火+身强 → 用神=水 → 水×阵营 → 观音/龙王等
- 火型人+最旺火+身弱 → 用神=木 → 木×阵营 → 文殊/月老等（不同！）

这才是加入身强身弱的真正价值——让一部分"同日主同最旺但不同体质"的人得到不同的守护神。
