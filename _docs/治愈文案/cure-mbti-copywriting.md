# 用神×MBTI 16型 五行缺口文案库（80条）

> **字数红线**：每条严格控制在50字以内（含固定开头）
> **术语洁癖**：零心理学黑话、零八字行话、零MBTI英文缩写
> **毒舌画风**：精准打中每个人格面具下的痛点，像一篇微型心理小说
> 
> **结构**：4阵营 × 4型 × 5缺口 = 80条
> **v3.0公式**：`你这辈子反复面对的题目：**缺{五行}**。{毒舌文案}`
>
> **哲学定位**：缺的五行不是"决定了你是谁"，而是"你反复面对的同一道题"。方法可以换，题目不会变。

## 附录A：八字逻辑校验记录（Bug修正）

> 以下是创作过程中发现并修正的八字逻辑Bug，供后续迭代参考。

### Bug #1：SJ·灯塔守望者·缺土
- **原版**：被死规矩的水泥封了双脚，挪不动
- **诊断**：「被封死」= 土旺极滞（土太多），缺土的核心是「没有承载力、缺乏地基、内心空虚没底气」
- **修正**：死守规矩装作稳如泰山，其实风一吹你就慌。你用刻板掩饰自己根本没有地基的灵魂。

### Bug #2：NF·旧书店里的诗人·缺水
- **原版**：把脆弱冻成冰块砸自己
- **诊断**：「冻成冰块」= 水极寒（水多无火），缺水的核心是「干枯、缺乏润滑与变通」，遇伤害直接碎裂而非冻结
- **修正**：灵魂干涸得结了壳。学不会像水一样化解伤害，任何一点磕碰都能让你直接四分五裂。

### Bug #3：SP·篝火旁的吉他手·缺水
- **原版**：顺着别人的情绪流浪，像泡沫，毫无定力
- **诊断**：「随风飘荡无定力」= 缺土或缺木症状。SP缺水 = 缺乏水的深邃与静谧，永远在狂躁因为内心干涸，不敢面对安静
- **修正**：永远在躁动，用疯狂的感官刺激掩饰内心的干涸。你根本没有能让自己静下来独处的深渊。
