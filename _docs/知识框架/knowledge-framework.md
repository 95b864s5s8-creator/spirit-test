## 〇、核心哲学：题目 × 方法（v3.0新增）

### 0.1 底层洞察

> **MBTI是动态策略，八字用神是永恒课题。**

每个人这辈子都在反复面对同一道题（用神缺的五行），而你的人格（MBTI）就是你现在正在用的解题方法。方法可能是你从小学会的，可能是环境逼出来的，可能好用也可能不好用——但题目不会变。

| | MBTI（人格） | 八字用神（缺的五行） |
|---|---|---|
| **本质** | 你应对世界的**工具箱** | 你一辈子反复遇到的**那道题** |
| **时间性** | 临时策略，随环境变化 | 永恒课题，反复出现 |
| **变化性** | 可能变，可能不变，取决于适应性 | 题目不变，变的是你的解法 |
| **比喻** | 你手里的锤子/螺丝刀/扳手 | 那颗你反复拧却拧不紧的螺丝 |
| **产品叙事** | "你现在的解题方法" | "你这辈子在反复面对的问题" |

### 0.2 这个洞察改变了什么

**旧版叙事**：你是XX人（固定标签）→ 你有XX问题（诊断式）→ 神来救你（被动拯救）
**新版叙事**：你这辈子在反复面对同一个问题（用神）→ 你现在用的解题方法是XX（动态策略）→ 这是你的能量补给站（主动赋能）

| 结果面 | 旧叙事 | 新叙事 |
|--------|--------|--------|
| **Face 1 你是什么人** | "你是XX人"（静态标签） | "你现在的解题方法是XX"（动态策略描述） |
| **Face 2 你的精神状态** | "你有XX问题"（诊断式） | "你正在用XX方法反复面对同一个问题"（叙事式） |
| **Face 3 守护神** | "神来救你"（被动拯救） | "这是你的能量补给站——这道题你不必一个人解"（主动赋能） |
| **Face 4 带走** | "买产品疗愈" | "调频道具——帮你的解题方法更适配这道题" |

### 0.3 产品免责话术（自然化解贴标签批评）

> "荣格说过，他的理论不是用来给人贴标签的。
> 我们也不是。
> 你看到的人格类型，不是'你是谁'——而是'你现在正在用哪种方式面对这个世界'。
> 方法可以换。但那道题，值得你认真对待。"

### 0.4 身强身弱的真实角色（v3.0修正）

> ⚠️ **重要修正**：v3.0初版错误地将「身强=方法有效 / 身弱=方法失效」。这是不对的。
> 身强身弱反映的是日主自身的能量储备，不是解题方法的优劣。

| 维度 | 身强 | 身弱 |
|------|------|------|
| **本质** | 日主能量充沛，根基扎实 | 日主能量不足，根基薄弱 |
| **底层作用** | 参与用神计算（能扛则用克泄耗） | 参与用神计算（扛不住则用生扶） |
| **用户展示** | 能量储备：满格 🔋 | 能量储备：需充电 🪫 |
| **不等于** | ❌ 不等于方法有效 | ❌ 不等于方法失效 |

**方法有效/失效的正确判定** = 用神（缺的五行）× MBTI阵营 → 代偿有效性矩阵

详见 `compensation-matrix.md`：
- 🟢 你的天性恰好能补足缺口 → 有效代偿（强行进化，极度疲惫但有效）
- 🔴 你的天性恰好回避了缺口 → 无效防御（饮鸩止渴，画地为牢）

### 0.5 50种终极形态的定位

50种终极形态（日主×强弱×最旺）描述的是：**你当前的能量状态和方法风格**。
- 身强+最旺 = 你能量充沛且某种力量极强 → 描述这种充沛带来的表现
- 身弱+最旺 = 你能量不足但某种力量极强 → 描述这种错配带来的困境

**注意**：50种形态是「你当前的状态描述」，不是「方法有效/失效的判定」。有效/失效的判定权在代偿矩阵手里。

**算法判断扩展**：50种形态不只是结果展示，还应作为题目设计的依据。如果某种形态对应的人群占比极大（如"水多木浮"型极多），说明这类"题目×方法错配"是当代高频困境，应在题目中有专门的探测题来识别。详见第三节3.2。

## 一、先天层：八字五行能量画像

### 1.1 数据输入

用户只需提供**出生年月日**，时辰为可选项（有时辰八字更准，但很多用户不知道自己的时辰，故设为选填）。

从出生日期推算：
- **年柱**：年份 → 天干地支
- **月柱**：月份 → 天干地支（简化：按节气月份）
- **日柱**：日期 → **日主天干**（核心）
- 日主天干的阴阳属性

### 1.3 五行能量画像（古籍校准版）

> 以下形貌性情描述综合《五行精纪·论形貌性情》与《三命通会·论五行》。

| 五行 | 核心特质 | 阳面（古籍原话） | 阴面（过载时） | 现代毒舌翻译 |
|------|---------|----------------|---------------|-------------|

### 1.4 五行生克关系（古籍校准版）

> 《子平真诠》："四时之运，生与克同用，克与生同功。譬如人之养生，固以饮食为生，然使时时饮食，而不使稍饥以待将来，人寿其能久乎？"

**相生**（补给）：木→火→土→金→水→木
**相克**（制约）：木→土→水→火→金→木

**关键校准——克制即是保护：**
- 木克土 → 不是土倒霉，是土不被水冲走（木扎根固土）
- 水克火 → 不是火灭了，是火不会烧完一切（水节制火）
- **MVP话术**："你缺的不是克制，是被人管着。连古人都能理解'克'是一种保护，你为什么觉得所有的规矩都是在针对你？"

**日主强弱判断**（简化版）：
- 看日主五行在出生月份是否"得令"（当令者旺）
- 看其他柱对日主是生扶还是克泄
- MVP简化：只看月令 + 日主天干，判断身强/身弱/平衡

### 2.2 维度说明（荣格原著+现代毒舌翻译）

#### 维度一：能量方向（E vs I）

> 荣格定义：外倾 = 心理能量指向外部客体；内倾 = 心理能量指向内部主体

| 极端E端 | 极端I端 |
|---------|---------|
| 在人群中充电，独处会电量告急 | 独处时充电，社交是在放电 |
| 表达欲强，想到什么就说 | 内心世界丰富，但"说出来就没意思了" |
| 行动先于思考（Se/Ne/Te/Fe驱动） | 思考先于行动（Si/Ni/Ti/Fi驱动） |
| **毒舌**：你不是社牛，你是没学会和自己相处 | **毒舌**：你不是深沉，你是没学会跟人说话 |

#### 维度二：信息处理（N vs S）

> 荣格定义：感觉 = 通过感官获取具体信息；直觉 = 通过无意识获取可能性

| 极端N端 | 极端S端 |
|---------|---------|
| 关注未来、可能性、抽象概念（Ni预见结局，Ne发散联想） | 关注当下、细节、具体事实（Se感官沉浸，Si经验对比） |
| 脑子里永远有100个Plan B，但没有Plan A | 有一个Plan A，而且执行到完美 |
| "这件事的未来潜力比现在更重要" | "你连现在都搞不定，谈什么未来" |
| **毒舌**：你活在未来，所以你现在一事无成 | **毒舌**：你活在当下，所以你永远活在别人的计划里 |

#### 维度三：决策方式（T vs F）

> 荣格定义：思维 = 依据逻辑因果做判断；情感 = 依据价值感受做判断

| 极端T端 | 极端F端 |
|---------|---------|
| 用逻辑分析做决策，对事不对人（Te追求效率，Ti追求自洽） | 用价值观和感受做决策，以人为本（Fe照顾群体，Fi忠于自我） |
| "这不合逻辑"是最常用的拒绝理由 | "这样不对劲"是最常用的判断依据 |
| 冷静但可能显得无情 | 温暖但可能失去原则 |
| **毒舌**：你不是理性，你是没心 | **毒舌**：你不是善良，你是没脑子 |

#### 维度四：生活方式（J vs P）【v2.0新增】

> Myers-Briggs扩展维度：判断 = 偏好结构化和决策闭合；感知 = 偏好灵活性和信息开放

| 极端J端 | 极端P端 |
|---------|---------|
| 事情要有计划、有进度、有终点 | 计划是用来打破的，惊喜才是生活的味道 |
| 提前做完才有安全感，deadline是终点线 | deadline才是第一生产力，凌晨两点效率最高 |
| 行李箱打包像乐高，每件东西有固定位置 | 行李箱打包像俄罗斯方块，使劲塞总能关上 |
| **毒舌**：你不是有计划，你是控制欲太强 | **毒舌**：你不是随性，你是懒得上心 |

### 2.3 功能轴对立（隐藏的诊断维度）

> 每个维度两端互为对立面（Beebe模型）。**越偏执一端，劣势端越脆弱——这是毒舌文案的核心切入点。**

| 功能轴 | 核心冲突 | 偏执时的表现 |
|--------|---------|-------------|
| Se ↔ Ni | 活在当下 vs 预见未来 | "你每天都在享受生活，但你的人生没有方向" / "你永远在规划未来，但你从未活在当下" |
| Si ↔ Ne | 经验传承 vs 探索可能 | "你不是稳重，你是害怕改变" / "你不是创新，你是三分钟热度" |
| Te ↔ Fi | 外部效率 vs 内在价值 | "你不是高效，你是把人当工具" / "你不是有原则，你是不会变通" |
| Ti ↔ Fe | 内部逻辑 vs 他人感受 | "你不是理性，你是杠精" / "你不是善良，你是讨好型人格" |

## 三、交叉模型：先天×后天的诊断逻辑

### 3.1 核心公式（v3.0升级）

```
诊断 = f(日主五行, 身强身弱, 最旺五行, IE, SN, TF, JP, Grip, 核心困境, 自我认知)
```

**v3.0重新定义输出结构（4面叙事 — 题目×方法框架）**：
> 用户看到的只有4面，零专业术语。详细模板见 `review-results.md` 第五节。

1. **你这辈子在反复面对什么问题**：出厂人设（日主本性）+ 用神（你缺的五行=那道题）+ 一句话点题
2. **你现在的解题方法是**：灵魂称号（MBTI人格）+ 阵营名 + 一句话标签 + 能量条 + 当前方法是否有效（50种终极形态）
3. **你的能量守护神**：主神+辅神+静心建议（保留，好玩）
4. **带走**：生成确诊单 + 推荐产品（调频道具，帮你的方法更适配这道题）

### 3.2 三位一体终极形态（v3.0升级：题目×方法视角）

> **50种终极形态 = 日主五行（5）× 身强/身弱（2）× 最旺五行（5）= 50种**

**v3.0核心逻辑**：你的出厂设置是什么（日主）+ 你的能量储备（强/弱）+ 你正在遭遇哪种最大的力量（最旺五行）= 你当前用什么方法解这道题。

- **身强 + 最旺** = 能量充沛+某种力量极强 → 描述这种充沛带来的表现（不等于方法有效）
- **身弱 + 最旺** = 能量不足+某种力量极强 → 描述这种错配带来的困境（不等于方法失效）

> ⚠️ **关键区分**：
> - 50种终极形态 = 你当前的**状态描述**（你是怎么用方法的）
> - 代偿有效性矩阵 = 你的方法的**有效性判定**（你用对了还是用错了）
> - 两者的关系：同一道题（同一个用神），50种形态的人可能用不同的方法解题，但有效/失效取决于方法是否匹配题目

**算法判断扩展说明**：
50种形态不只是结果标签，还应是**题目设计的反馈信号**。如果测试数据中某些形态占比极高（如"水多木浮""火多木焚"等身弱困境态），说明当代青年中"想太多做太少""燃烧自己照亮别人"是高频困境。此时应：
1. 在题目中增加针对该困境的专门探测题（如"你最近是不是总觉得自己想了很多但一个都没做"）
2. 在结果页中对高频困境形态给予更精细的解读（不仅说"你的方法失效了"，还说"你具体卡在哪里"）
3. 对应推荐更精准的能量处方和产品

**未来可扩展**：通过回收测试数据，统计50种形态的分布，动态调整题目权重和探测维度。

完整50种形态文案见 `review-results.md` 第一节。

## 五、毒舌文案风格指南

### 5.1 核心原则

1. **先扎心，再给糖** — 先描述痛苦（用户会疯狂点头），再给解决方案
2. **用"你"开头** — 不是"木型人容易…"，是"你就是那种…的人"
3. **具体场景化** — 不是"性格固执"，是"开会时别人说方案不行，你表面点头心里已经在想怎么证明他错"
4. **适度自嘲** — "恭喜你，你是一个行走的矛盾体"
5. **留悬念** — 结果页不直接说完整，引导分享/付费

### 5.2 文案级别对照

| 级别 | 标题风格 | 内容风格 | 示例 |
|------|---------|---------|------|
| 轻度 | 调侃式 | 轻松幽默 | "你这个人吧，优点是稳定，缺点也是稳定" |
| 中度 | 扎心式 | 一针见血 | "你不是佛系，你是懒得挣扎了" |
| 重度 | 地狱笑话式 | 毒舌但不恶毒 | "佛祖看到你的八字都沉默了三秒" |

### 5.3 禁忌

- 不涉及真正的算命/预测未来（避免玄学合规风险）
- 不制造焦虑后不给解决方案
- 不贬低任何宗教信仰
- 不用过于专业的命理术语（用户看不懂）

### 8.1 日柱推算算法

出生日期 → 日柱天干的算法：
- 以已知基准日（如2000年1月1日=甲子日）为锚点
- 计算目标日期与基准日的天数差
- 天干 = (基准天干序号 + 天数差) % 10
- 地支 = (基准地支序号 + 天数差) % 12
