# 结果类型对齐文档 v7 — 三位一体终极框架（题目×方法版）

> **核心升级（v3.0）**：从「50种终极形态=你的状态」升级为「50种终极形态=你当前用什么方法解这道题」。
>
> **底层哲学**：MBTI是动态策略（工具箱），八字用神是永恒课题（那道反复出现的题）。人格不是"你是谁"，而是"你当前用什么方法面对世界"。方法可以换，但题目不会变。
>
> **计算四步走**：
> 1. 算身强身弱（同党得分 vs 异党得分，50分界线）→ 用于计算用神 + 能量储备指标
> 2. 算五行极值（八字中木火土金水各自总得分，找最高的那个五行）
> 3. 算用神（缺的五行）→ 你反复面对的那道题
> 4. 匹配代偿矩阵：用神（缺的五行）× MBTI阵营 → 有效代偿/无效防御
> 5. 匹配终极形态：日主五行 + 强弱状态 + 最旺五行 → 50条文案中的1条（你当前的状态描述）
>
> **结果组合**：50种终极形态（状态描述）× 代偿矩阵（有效/无效判定）× 16种性格标签（解题风格）× 4种Grip警告（方法失灵模式）

## 一、三位一体终极形态（50种 — 核心结果层）

> **v3.0 核心逻辑**：你的出厂设置是什么（日主）+ 你的能量储备（强/弱）+ 你正在遭遇哪种最大的力量（最旺五行）= **你当前用什么方法解这道题**。
>
> **身强身弱的真实角色**：
> - 身强 = 能量充沛，根基扎实 → 参与用神计算（能扛则用克泄耗）
> - 身弱 = 能量不足，根基薄弱 → 参与用神计算（扛不住则用生扶）
> - ⚠️ **身强身弱不等于方法有效/无效**。有效/无效由代偿矩阵（用神×MBTI阵营）判定
>
> **用神（缺的五行）= 你反复面对的那道题**。它不随时间改变，不随环境改变——它是你出厂设置里最核心的命题。

## 附录B：身强身弱判断逻辑（计分权重法）

### 五行映射

| 五行 | 天干 | 地支（本气） |
|------|------|-------------|
| 木 | 甲、乙 | 寅、卯 |
| 火 | 丙、丁 | 巳、午 |
| 土 | 戊、己 | 辰、戌、丑、未 |
| 金 | 庚、辛 | 申、酉 |
| 水 | 壬、癸 | 亥、子 |

### 同党与异党

- **同党（加分项）**：同类（比劫）+ 生我者（印枭）
- **异党（减分）**：我克者（财）+ 克我者（官杀）+ 我生者（食伤）

例：日主甲木 → 同党=木+水，异党=土+金+火

### 计分规则

| 位置 | 同党得分 | 说明 |
|------|---------|------|
| 月支（出生月地支） | +40分 | 同党=得令，异党=+0分（失令） |
| 其他地支（年/日/时支） | 每个+12分 | |
| 其他天干（年/月/时干） | 每个+8分 | |

### 判定

- **总分 ≥ 50** → 身强 | **总分 < 50** → 身弱

## 附录C：五行极值算法

### 权重分配（总分108分）

| 位置 | 分值 | 说明 |
|------|------|------|
| 月支（月令） | 40分 | 绝对C位，决定气候和主导力量 |
| 年支、日支、时支 | 各12分 | 3×12=36分 |
| 年干、月干、日干、时干 | 各8分 | 4×8=32分（日干也算8分计入） |

### 计算步骤

1. 初始化 `scores = {木:0, 火:0, 土:0, 金:0, 水:0}`
2. 遍历八字8个位置，查五行映射表，累加对应分数
3. `Math.max(scores)` 得出最旺五行
4. 用最旺五行与日主的关系匹配病态类型

### 平局处理（Tie-breaker）

1. **月令优先**：并列中谁占了月支谁赢
2. **异党优先**：都没占月令时，日主的异党（克/泄/耗）优先
3. **天干透出优先**：同为异党时，天干出现的那个赢

### 案例演示：戊辰 庚申 甲寅 癸酉

| 位置 | 字 | 五行 | 分值 |
|------|----|------|------|
| 年干 | 戊 | 土 | 8 |
| 年支 | 辰 | 土 | 12 |
| 月干 | 庚 | 金 | 8 |
| 月支 | 申 | 金 | **40** |
| 日干 | 甲 | 木 | 8 |
| 日支 | 寅 | 木 | 12 |
| 时干 | 癸 | 水 | 8 |
| 时支 | 酉 | 金 | 12 |

- 木：8+12=20 | 火：0 | 土：8+12=20 | 金：8+40+12=**60** | 水：8
- 日主=木，同党(木20+水8)=28 < 50 → **身弱**
- 最旺=金(60)，金克木 → 触发 **【金锐木折】**
