# 结果页 v4.0 — 重构方案

> 日期：2026-04-20
> 基于：代偿有效性矩阵v1.0 + 17神守护神矩阵v2 + SBTI传播研究

## 一、核心问题

### 1.1 守护神与代偿判定断裂

当前结果页的守护神和代偿判定是两个独立模块，用户看完"你的方法在伤害你"后看到"你的守护神是观世音"，两者之间没有逻辑衔接。

### 1.2 页面信息流倒置

当前Face1把守护神放第一屏，用户还没知道自己缺什么就被塞了一个守护神。好比医生没问诊就开药。

### 1.3 传播路径错误

当前依赖"分享卡"作为传播载体，但SBTI证明：**截图传播力 >> 分享卡传播力**。用户不会点"生成海报"按钮，他们会直接截图。分享卡每多一步操作，流失率指数级上升。

## 二、守护神叙事重构

### 2.1 核心公式：守护神 = 代偿的解药

逻辑链：

```
你的出厂设置（缺金）
  → 你的天性选择的解题方法（NF的温柔包容）
  → 系统判定：这个方法在伤害你（🔴无效防御）
  → 所以你需要一种你不会自然选择的力量
  → 守护神（文殊菩萨的般若之剑）
```

守护神不再是独立推荐，而是代偿判定的**必然结论**。

### 2.2 每格拆🟢/🔴两版文案

| 代偿结果 | 用户状态 | 神的角色 | 文案方向 |
|---------|---------|---------|---------|
| 🟢有效代偿 | 你扛住了但很累 | 减负/休整型 | "你靠爱活过来了，但也别把所有生命力都拴在别人身上" |
| 🔴无效防御 | 你的方法在伤害你 | 破局/换武器型 | "你的温柔正在变成你的软肋，你需要文殊菩萨那把般若之剑" |

**20格 × 2版本 = 40条结论文案**（待撰写）

### 2.3 神的数量和人选

**不动**。17位覆盖20格合理，16位主力+1位Tie-Breaker。

## 三、页面结构重构

### 3.1 四屏信息流（每屏独立可截图）

```
【第一屏】身份认同
  大emoji + 称号 + 毒舌金句
  → 用户笑了，觉得准，愿意继续看
  → 截图价值：⭐⭐⭐⭐（身份标签+好笑金句）

【第二屏】出厂设置
  处方表格 + 终极形态文案（50种之一）
  → 用户知道自己"缺什么"+"当前状态"
  → 截图价值：⭐⭐⭐（精准诊断感）

【第三屏】代偿判定（扎心暴击）
  🟢有效代偿 / 🔴无效防御 + 扎心文案
  → 用户被击中："对，我就是这样的"
  → 截图价值：⭐⭐⭐⭐（情绪爆点）

【第四屏】守护神揭晓（压轴）
  大emoji + 光晕 + 神名
  从代偿判定自然引出："所以你需要XX"
  这是全页情绪最高点
  → 截图价值：⭐⭐⭐⭐⭐（好奇驱动+敬畏感）

【底部】分享+CTA
  分享卡（可选，给愿意多操作的人）
  CTA按钮（测测你的守护神/推荐朋友测）
```

### 3.2 情绪曲线

```
笑（第一屏）
  → 嗯准（第二屏）
    → 扎心（第三屏）
      → "原来如此"（第四屏）
        → 转发（底部）
```

对比旧结构：

| | 旧结构 | 新结构 |
|--|--------|--------|
| 第一反应 | "观世音？为什么？" | "哈哈哈这说的不就是我" |
| 情绪曲线 | 平→平→平→暴击 | 笑→准→扎心→顿悟→转发 |
| 守护神定位 | 开头硬塞 | 压轴揭晓 |
| 代偿→守护神 | 断裂 | 自然衔接 |

## 四、传播策略

### 4.1 核心洞察（来自SBTI研究）

> **用户不是在分享测试结果，是在分享"帮我开口"的那个东西。**

SBTI做对的事：
- 结果不是报告，是**梗**
- 轻微冒犯 > 夸奖（转发力高10倍）
- 替用户开口（用户不好意思说的，测试帮他说了）
- 结果=截图=社交货币，天然为被截图而生

### 4.2 截图优先，分享卡补充

| | 截图 | 分享卡 |
|--|------|--------|
| 操作路径 | 1秒 | 3-5步 |
| 传播阻力 | 几乎为零 | 每步流失率指数级上升 |
| 朋友圈原生感 | 真实 | 像广告 |
| 内容控制 | 产品方无法控制 | 完全可控 |

**主要传播路径 = 截图**，不是分享卡。

### 4.3 每屏都有独立传播价值

SBTI的信息量简单到一屏装得下，我们的信息量装不下。所以**让每一屏都有独立的截图价值**，而不是指望用户看完4屏后在最后生成一张分享卡。

| 屏 | 朋友看到截图的反应 | 想来测的动力 |
|---|------------------|------------|
| 第一屏 | "哈哈哈这说的谁" | 身份认同 |
| 第二屏 | "有点准" | 想知道自己的诊断 |
| 第三屏 | "卧槽我可能也是这样" | 想知道自己的判定 |
| 第四屏 | "守护神？什么守护神" | 好奇驱动 |

### 4.4 分享卡的设计原则

分享卡作为**补充选项**（底部可选），不作为唯一传播路径：

```
┌─────────────────────────────┐
│  📋 测了吗                   │
│  🎤 全网最准的互联网嘴替      │
│  「你活成了所有人的心理咨询师，│
│    但谁来咨询你」              │
│  🔴 你的温柔正在变成你的软肋    │
│  🪷 守护神观世音有个秘密       │
│  [测测你的守护神是谁 →]        │
└─────────────────────────────┘
```

分享卡三要素：身份标签 + 毒舌金句 + 好奇钩子
