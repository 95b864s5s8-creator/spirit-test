## 🔴 必须改：第19题（核心困境）

### 当前问题

| 问题 | 说明 |
|------|------|
| 4选项对齐不了17神 | 老分类（输出/输入/压力/迷失）只有4档，17神需要更细的功能分类 |
| 状态描述共鸣度不够 | "每天忙到飞起"是外在状态，不如内心独白直接 |
| Tie-Breaker能力弱 | "输出过度"和"压力摧残"边界模糊，很多人既忙又被催 |
| 选项数限制 | 不再锁死4个，可以扩展到5-6个 |

### 改造方案

**5选项版（推荐）**：按"用户当前最想解决的事"分——这个维度更直觉，直接对应守护神功能：

| 选项 | 内心独白 | 对应神的功能方向 |
|------|---------|-----------------|
| A | "我太累了，付出太多回报太少，没人看见" | 承载型：地藏/妈祖（需要被托住） |
| B | "我脑子停不下来，越想越焦虑，但行动跟不上" | 破迷型：文殊/太上老君（需要减法/变通） |
| C | "我知道该做什么，就是迈不动腿" | 行动型：哪吒/关公（需要被推一把） |
| D | "关系里我总是付出更多的那个，但好像没人珍惜" | 边界型：月老/悟空（需要学会收回） |
| E | "说不上哪里不对，就是觉得活着没劲" | 破局型：雷神/龙王（需要一次震碎） |

**6选项版（备选）**：如果需要对齐更细，可以再加一个F：

| F | "我总觉得别人比我轻松，我在拼命但其实不知道为了什么" | 疑问型：达摩/观音（需要觉醒/照见真相） |

## 🟡 建议微调：6题

### 第8题 SN② 消费决策

**当前**：A"先看测评对比参数查Reddit讨论，最后买了个完全不同的东西"
**问题**：偏极端搞笑，部分N用户会觉得在吐槽自己，可能选C
**建议**：A改为"先看测评对比参数查讨论帖，纠结三天，最后买了个完全不同但更有创意的东西"

### 第11题 TF② 网上不同观点

**当前**：B"想了想，每个人立场不一样，算了，划走"
**问题**："立场"是F的价值观暗示，部分T用户也划走但原因不同（觉得不值得争论）
**建议**：B改为"想了想，争论下去没意义，划走"

### 第20题 认知偏差

**当前**：3个选项都预设用户"在社交平台发内容"
**问题**：有一类用户根本不发朋友圈/小红书
**建议**：加D选项"我不怎么发朋友圈，别人眼中的我是个谜，连我自己都不知道"（cognition: 'invisible'）
**或不改**：这类用户可以选C（懒得立人设），逻辑也通

### 第21题 交叉IE×SN 万字长文

**当前**：A同时投E和N，B同时投I和S
**问题**：E+S（看完转发但觉得过度分析没意义）和I+N（看完默默思考不转发）被忽略
**结论**：作为交叉辅助题，可以接受。但A选项偏"E+N"刻板印象
**建议**：不改（1票辅助验证，影响有限）

### 第24题 IE⑥ 新环境

**当前**：主动搭话 vs 找角落等别人来
**问题**：和题5（饭局十人）场景高度重叠——都是"陌生人环境"，核心行为模式一样
**建议**：换场景。
- 方案1（线上）：新加入一个微信群，A"第一周发10条消息比老成员还活跃" / B"潜水三个月第一条消息是帮别人砍一刀" / C"默默观察一个月才第一次冒泡"
- 方案2（线上会议）：A"开摄像头主动发言" / B"关摄像头关麦" / C"开了摄像头但关了脸只露背景"

### 第26题 SN⑤ 看说明书

**当前**：A"不看说明书，先自己摸索"
**问题**：不看说明书也可能是S（用过类似产品凭经验），不一定是N
**建议**：A改为"不看说明书，凭直觉乱按，结果发现了一种说明书上没写的用法"——更精准指向N的直觉+发散
