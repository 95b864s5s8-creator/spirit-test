# 「测了吗」30道题完整框架 v3.0

> 守护神 = 八字用神(出生日期) × 矩阵(16型阵营)
> 用户感知：30道题 + 出厂日期 → 共同算出守护神

## 整体架构

### 输入层（2个独立数据源）

| 数据源 | 题号 | 产出 | 用途 |
|--------|------|------|------|
| 出生日期 | 第1题 | 八字四柱 → 用神五行（5选1） | 矩阵X轴 |
| 29道选择题 | 第2-30题 | 荣格4维(IE/SN/TF/JP) → 16型 → 阵营（4选1） | 矩阵Y轴 |

### 关系圈层结构（6层 × 5道 = 30道）

用户做完30道题的感受："这不只是一个测试，这是把我整个人扒干净了"

| 章节 | 关系层 | 题数 | 核心探索 | 进度氛围 |
|------|--------|------|---------|---------|
| 第一章 | 🪞 人与自己 | 6道 | 你独处时是什么样、内心世界、自我认知 | "准备好了吗，先照照镜子" |
| 第二章 | 💕 人与爱人 | 5道 | 恋爱模式、亲密关系中的你、情感需求 | "别紧张，不问择偶标准" |
| 第三章 | 🤝 人与朋友 | 5道 | 社交模式、边界感、你给朋友的印象 | "朋友才是照妖镜" |
| 第四章 | 👨‍👩‍👧 人与家人 | 4道 | 原生家庭、代际关系、你的成长痕迹 | "这一章可能有点扎" |
| 第五章 | 🏙️ 人与社会 | 5道 | 职场、规则、陌生人、你在这个世界的位置 | "你和社会的关系，微妙吧" |
| 第六章 | 🌌 人与未知 | 4道 | 压力反应、存在主义、终极困境、灵魂深处 | "最后一章，深呼吸" |

**总计：1道出生 + 29道选择 = 30道**

### 维度分配策略

守护神只需要：**SN(≥2) + TF(≥2) + 困境(1) = 最少5道**
展示层还需要：**IE(≥2) + JP(≥2) + Grip(1) = 5道**
其余18道全部服务于"关系全景扫描"的沉浸感，同时给各维度补充投票增加精度

| 维度 | 最少需要 | 实际分配 | 题目编号 |
|------|---------|---------|---------|
| SN | 2 | **5道** | 第2/4/8/18/22题 |
| TF | 2 | **5道** | 第3/9/12/20/25题 |
| IE | 2 | **5道** | 第5/6/14/15/19题 |
| JP | 2 | **4道** | 第7/11/16/23题 |
| Grip | 1 | **1道** | 第27题 |
| 困境 | 1 | **2道** | 第28/29题 |
| 认知偏差 | 0 | **1道** | 第30题 |
| 多维交叉 | 0 | **3道** | 第10/17/26题（同时投2个维度） |
| 无维度 | 0 | **3道** | 第13/21/24题（纯氛围/破冰/过渡） |
| **总计** | **10** | **30** | |

> 多维度题的妙用：一道题同时给IE+JP投票（交叉题），信息密度翻倍

#### 第26题：深夜思考（IE × SN 交叉）
```
第 26 题 | 🌌 人与未知

凌晨两点还醒着，你大概率在想——

A. "如果当初选了那条路，现在会在哪里？"——脑子里跑马灯
B. "明天早上吃什么？"——务实的人有务实的失眠
C. 不在想任何具体的事，就是睡不着，看着天花板发呆
```
- **维度**：交叉题（SN × IE）
- **答案**：A→N+I / B→S+I（务实但也是内省） / C→null
- **补充说明**：凌晨两点的想法最能暴露底层认知模式

#### 第27题：压力反应（Grip探测）
```
第 27 题 | 🌌 人与未知

最近一次觉得"快撑不住了"的时候，你最接近——

A. 暴饮暴食/疯狂下单/刷短视频到凌晨——用感官刺激填满自己
B. 反复回想最近哪里出了问题，翻跟所有人的聊天记录找线索
C. 对身边所有人说话都变得很冲，"你能不能别烦我"
D. 关掉手机谁都不理，一个人待着，不想被任何人找到
```
