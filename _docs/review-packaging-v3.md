# 包装文案重构方案 v3

## 一、调研核心结论

### SBTI为什么火
- **名字即传播**：谐音梗自带话题
- **结果不追求准确，追求共鸣**：「吗喽」「死者」——用户看完第一反应是「怎么这么像我」
- **只做最短闭环**：测试→结果→分享，无注册/付费/广告
- **轻微冒犯感=高传播力**：像朋友吐槽，边损边懂
- **不定义你是什么人，陪你一起吐槽**

### 我们的优势（vs SBTI）
SBTI没有真正底层逻辑，纯AI生成玩梗。我们有：
1. **八字×荣格**双引擎，真的能算出东西
2. **50种终极形态**，每个人都有独特结果
3. **8位守护神**，天然有分享话题
4. **佛教文创**品牌调性，天然跟玄学/疗愈挂钩

### 我们当前的问题
1. **想表达的太多**——4面结果页塞了：出厂设置+毒舌定位+终极形态+认知偏差+Grip弱点+守护神+静心建议+商业转化。用户看完累。
2. **术语没清干净**——「成就态」「困境态」「认知偏差检测」还在
3. **守护神藏太深**——放在Face3，很多人可能懒得翻到那页
4. **缺少"一句入魂"的定位**——用户分享时不知道怎么推荐这个测试

## 三、结果页重构：4面→3面，每面一个核心信息

### 现有4面的问题
| 面 | 现在的内容 | 问题 |
|----|-----------|------|
| Face1 | 阵营+称号+出厂设置+毒舌 | 信息过载，术语残留 |
| Face2 | 终极形态+认知偏差+Grip弱点 | 三块内容堆砌，没有节奏 |
| Face3 | 守护神+静心建议 | 守护神是最大卖点但藏太深 |
| Face4 | 分享卡+商业转化 | 太硬 |

### 新3面结构

**第1面：你是什么怪物（身份认同页）**
核心目的：让用户觉得「这说的就是我！」然后想截图分享

内容：
- 阵营标签（想太多组/太懂事组/能扛事组/管不住组）
- 灵魂称号（下棋的人/精神上的扫地机器人...）
- 一句毒舌金句（精准扎心）
- 你的出厂属性（木/火/土/金/水 × 充沛型/蓄力型）← 用emoji+一句话，不展开

**第2面：你的守护神是谁（好奇心驱动页）**
核心目的：让用户想截图发给朋友「我的守护神居然是XX」

内容：
- 主守护神（大卡片，名字+emoji+一句话为什么选他）
- 辅守护神（小卡片）
- 一句调侃（结合守护神特性+用户灵魂称号）
- 你的灵魂病态（终极形态名+一句话概括，不是完整解释）

**第3面：带走（转化+传播页）**
核心目的：分享+商业引导

内容：
- 精简版分享卡（称号+守护神名+毒舌金句）
- 一键分享按钮
- 疗愈产品推荐（一句话+按钮）
- 调侃式结尾（比如「此报告由玄学CT机自动生成，不构成医学建议」）

### 被砍掉的内容（但可以保留在数据里）
- 完整的「终极形态」解释（explanation/summary大段文字）→ 砍
- 「认知偏差」详细分析 → 砍
- 「Grip弱点」详细分析 → 砍
- 「静心建议」→ 砍

> **核心原则**：用户看到的每一个字，都要么让他笑、要么让他认同、要么让他想分享。不能有"正确但无聊"的信息。

## 四、具体文案规范

### 禁用词清单
- ❌ 成就态、困境态
- ❌ 认知偏差
- ❌ Grip / 劣势功能
- ❌ 日主天干、四柱八字
- ❌ 身强身弱（改为"能量充沛型"/"蓄力待发型"已有）
- ❌ 终极形态（改为更口语的说法）

### 语气规范
- 像**聪明的损友**在给你做年度总结，不像心理医生在给你做咨询
- 每个区块最多2句话，超过就是废话
- 金句优先：能用一句说完的，绝不用两句

### 守护神推荐文案公式
> 基于「[灵魂称号]」的灵魂特质，[主神]被派来帮你——[一句调侃]
>
> 例：基于「下棋的人」的灵魂特质，**孙悟空**被派来帮你——你脑子里的那些惊天计划，需要一个敢第一个冲的人。
